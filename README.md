
# CareerTrack – Job Application Tracker (MERN)

A full-stack web application to track and manage job applications with real-time status updates and analytics.

## Features

- User Authentication (JWT-based login & register)
- Add, Update, Delete job applications
- Dashboard with real-time statistics
- Track company, role, and application status
- Fast and responsive UI

## Tech Stack

**Frontend**
- React.js
- Axios

**Backend**
- Node.js
- Express.js

**Database**
- MongoDB Atlas

## Dashboard Features

- View all applications
- Update status (Applied, Interview, Rejected, Offer)
- Delete applications
- Real-time stats using MongoDB aggregation
